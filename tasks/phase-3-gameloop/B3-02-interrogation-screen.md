## 概要

要件 §11.4 の尋問画面。対象選択 → 質問タイプ選択 → (証拠選択) → 実行 → 回答表示 → 信頼度変化アニメーション。

## 受け入れ条件

- [ ] `/play/[gameId]/interrogate` ルート
- [ ] 左: キャラクターリスト (生存のみ、信頼度バー付き)
- [ ] 中央: 選択中キャラの会話履歴 (尋問の往復)
- [ ] 右: 質問アクションパネル (5 タイプ、コスト表示)
- [ ] 証拠提示時は B2-06 と連動した modal で証拠選択
- [ ] 矛盾追及時は pin 一覧からピンを選択
- [ ] 実行ボタン → callSubmitInterrogation → 回答を順次表示
- [ ] 信頼度バーがアニメーションで増減

## 実装メモ

- target が選ばれていない時はアクションパネル disabled
- 1 回の応答中はボタン disabled (二重送信防止)
